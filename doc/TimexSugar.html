    before(span, date)

    Specs

    before(span :: TimexSugar.Span.t, date :: Timex.DateTime.t) :: Timex.DateTime.t

    Return a DateTime that is earlier than the given datetime by the given span.

    Examples

    iex> span = Elixir.TimexSugar.years(5)
iex> date = Timex.Date.from({{2015, 6, 24}, {14, 27, 52}})
iex> Elixir.TimexSugar.before(span, date)
%Timex.DateTime{calendar: :gregorian, day: 24, hour: 14, minute: 27, month: 6,
 ms: 0, second: 52,
 timezone: %Timex.TimezoneInfo{abbreviation: "UTC", from: :min,
  full_name: "UTC", offset_std: 0, offset_utc: 0, until: :max}, year: 2010}

iex> import TimexSugar
iex> date = Timex.Date.from({{2015, 6, 24}, {14, 27, 52}})
iex> 5 |> years |> before(date)
%Timex.DateTime{calendar: :gregorian, day: 24, hour: 14, minute: 27, month: 6,
 ms: 0, second: 52,
 timezone: %Timex.TimezoneInfo{abbreviation: "UTC", from: :min,
  full_name: "UTC", offset_std: 0, offset_utc: 0, until: :max}, year: 2010}
